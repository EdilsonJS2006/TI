Porta XNOR - O Comparador de Igualdade

Entenda a porta que detecta quando as entradas são iguais - a negação do XOR

O Que é uma Porta XNOR?

A porta XNOR (NÃO OU Exclusivo) é uma porta lógica especializada que implementa a negação da porta XOR. Ela detecta quando as entradas são iguais, produzindo saída 1 quando as entradas são idênticas (ambas 0 ou ambas 1). A porta XNOR é essencialmente um comparador de igualdade.

DEFINIÇÃO: A porta XNOR produz uma saída com valor lógico 1 (ALTO/VERDADEIRO) quando as entradas são IGUAIS. Para duas entradas, a saída é 1 quando ambas são 0 OU ambas são 1. A saída é 0 quando as entradas são diferentes.

Na álgebra booleana, a operação XNOR é representada pelo símbolo ⊙ ou ≡. A expressão booleana para uma porta XNOR de duas entradas é escrita como Q = A ⊙ B ou Q = (A ⊕ B)' ou Q = A·B + A'·B'.

XNOR

A porta XNOR é fundamental em circuitos comparadores e em sistemas de detecção de paridade. Ela é usada para verificar se dois bits são iguais, o que é essencial em operações de comparação, verificação de paridade e em certos tipos de somadores binários.

Propriedades Booleanas da Porta XNOR

Identidade: A ⊙ 0 = A' | A ⊙ 1 = A

Reflexão: A ⊙ A = 1 | A ⊙ A' = 0

Comutatividade: A ⊙ B = B ⊙ A

Associatividade: (A ⊙ B) ⊙ C = A ⊙ (B ⊙ C)

Relação com XOR: (A ⊙ B) = (A ⊕ B)'

Expansão: A ⊙ B = A·B + A'·B'
