Porta NOR - A Outra Porta Universal

Conheça a porta que nega a operação OR e também pode construir qualquer circuito digital

O Que é uma Porta NOR?

A porta NOR (NÃO OU) é uma porta lógica fundamental que combina uma porta OR seguida por um inversor (NOT), implementando a negação da disjunção lógica. Assim como a porta NAND, a porta NOR também é uma "porta universal" - qualquer outra porta lógica pode ser construída apenas com portas NOR.

DEFINIÇÃO: A porta NOR produz uma saída com valor lógico 1 (ALTO/VERDADEIRO) somente quando TODAS as suas entradas são 0. Se pelo menos uma entrada for 1, a saída será 0.

Na álgebra booleana, a operação NOR é a negação da operação OR. A expressão booleana para uma porta NOR de duas entradas é escrita como Q = (A + B)' ou Q = NOT(A OR B).

NOR

A porta NOR é tão importante quanto a porta NAND na eletrônica digital. Por ser uma porta universal, circuitos complexos podem ser implementados usando apenas portas NOR. Esta porta é particularmente útil em memórias estáticas (SRAM) e em certos tipos de flip-flops.

Propriedades Booleanas da Porta NOR

Neutralidade: (A + 0)' = A' | (A + 1)' = 0

Idempotência: (A + A)' = A'

Complemento: (A + A')' = 0

Comutatividade: (A + B)' = (B + A)'

Associatividade: ((A + B) + C)' = (A + (B + C))'

Lei de De Morgan: (A + B)' = A' · B'
